## Installation

1. Download `BAHAYKUBOAPP.apk` on an Android device.
2. Open the downloaded file.
3. If Android blocks the install, allow installation from unknown sources for the browser or file manager.
4. Tap install.
5. Open the Bahay Kubo App after installation.
